# Nexvy Salon Demo — Agent Instructions

## What this is
nexvy-salon-demo — A Next.js demonstration app for salon services by Nexvy.

## Tech stack
- Next.js (App Router)
- React 19
- Tailwind CSS v4
- TypeScript
- Lucide React for icons

## Design system
- Use Tailwind classes for styling (v4).
- Clean, modern salon aesthetics.
- Mobile-first responsive design.

## Hard rules — never break these
- Use React Server Components by default; only use `'use client'` when hooks or interactivity are required.
- Maintain high accessibility standards and good semantic HTML.
- Component-driven architecture using `src/components`.
- Consistent use of TypeScript types/interfaces for props.
